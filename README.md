# Geometric Studio — Precision in Imagination

Geometric Studio is a Cairo-based, globally-focused creative powerhouse engineering cinematic 3D experiences that bridge the gap between imagination and physical reality.

## 🚀 Capabilities

- **Motion Graphics**: High-impact visual sequences and broadcast-ready animations.
- **Product Visualization**: Precision-rendered 3D models for marketing and design validation.
- **Booth & Exhibition**: Architectural-scale environmental experiences for global events.
- **Anamorphic Screens**: State-of-the-art 3D illusions for forced-perspective digital out-of-home (DOOH).
- **Simulations**: Physically accurate fluid, cloth, and particle simulations.
- **AI Content**: Strategic integration of generative AI tools for enhanced creative workflows.

## 🛠 Tech Arsenal

Our technical stack is optimized for maximum performance and visual fidelity:

- **Core**: React, TypeScript, Vite, Framer Motion
- **Creative**: Blender, Unreal Engine, Cinema 4D, Houdini, After Effects
- **UI Architecture**: Tailwind CSS, shadcn-ui, Radix UI

## 💻 Development

### Prerequisites

- Node.js (v18+)
- npm / bun

### Setup

```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🏗 Build & Deployment

```bash
# Generate production bundle
npm run build

# Preview production build
npm run preview
```
